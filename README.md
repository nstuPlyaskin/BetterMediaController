# Управление медиа плеером по ГОСТу

Расширение **Chromium** (Brave, Chrome, MV3): на страницах с `<video>` / `<audio>` **стрелки** и **пробел** всегда означают перемотку, громкость и паузу — даже если у сайта свои шорткаты, другой фокус или несколько плееров.

| Клавиша | Действие |
|--------|----------|
| ← → | ±N c (по умолчанию 5) |
| ↑ ↓ | ±громкость (по умолчанию 5%) |
| Пробел | Пауза / play |

В полях ввода перехват **выключен**. Настройки: шаги, «только с Alt», чёрный список сайтов; **popup** — вкл/выкл для текущего сайта.

## Установка

1. [brave://extensions](brave://extensions) или [chrome://extensions](chrome://extensions) → **Режим разработчика** → **Загрузить распакованное**.
2. Папка **`dist`** (не корень репозитория).
3. После правок — **Обновить** на карточке расширения.

**Четыре** команды в манифесте с готовым сочетанием — лимит Chrome. Команду **«Пауза»** один раз назначьте в **Сочетания клавиш** на странице расширений (например Alt+Shift+P). Дубли **Alt+Shift+стрелки** — запас, если страница перехватывает клавиши раньше.

**Права:** `storage`, `activeTab`, `<all_urls>` — только работа на сайтах и настройки; **сбор данных на сервер нет**.
